Respecto a la privacidad y las cookies

  • Usamos el servicio de Clicky para tener estadísticas del número de visitas
  • Usamos el servicio de GitHub Pages para alojar esta página
  • Usamos el servicio de Bing para generar la isodistancia, pero a traves de nuestro propio intermediario, de modo que Bing, no puede en ningún caso conocer la ubicación u otros datos del usuario
  • Usamos Netifly como intermediario entre el navegador y el servicio de Bing, así que a Netifly le llegaran peticiones la posición
  • Aproximamos la posición del usuario a "puntos fijos" con un radio de unos 50m. Esto disminuye la precisión de los cálculos pero aumenta un pelín la privacidad
  • Guardamos en el navegador, ninguna otra página tiene acceso la posición y el cálculo de la isodistancia. Ninguna otra página salvo tendrá acceso a esos datos y nos permite mantener el servicio sin costes excesivos
